# Multi-frequency dynamic modelling of the French sovereign yield curve

This Github project focuses on the code underlying Laurent E. Calvet, Adlai J. Fisher, and Liuren Wu's paper *Staying on Top of the Curve: A Cascade Model
of Term Structure Dynamics*.

We are a group of 4 student from Ecole Polytechnique working under the supervision of Laurent-Emmanuel Calvet on the adaptation of this model to French sovereign yields.
